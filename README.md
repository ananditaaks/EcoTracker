**EcoTracker** 🌱

## Overview
EcoTracker is a **Java-based web application** that helps users track their **carbon footprint** based on daily activities such as **transportation**, **food consumption**, and **electricity usage**. 
It calculates **CO₂ emissions** and provides simple suggestions to promote **sustainable habits**.

## Features
- User authentication (Signup/Login)
  
- Activity tracking:
  - Transportation
  - Food consumption
  - Energy usage
    
- CO₂ emission calculation
- Dashboard with emission summary
- Basic sustainability tips

## Tech Stack
- Frontend: HTML, CSS, JavaScript, JSP
- Backend: Java Servlets, JDBC
- Database: MySQL
- Server: Apache Tomcat
- Deployment: Render, Railway

## Architecture
The project follows the **MVC (Model-View-Controller)** pattern:
- JSP → View (user interface)
- Servlets → Controller (handles requests and logic)
- DAO/JDBC → Model (database interaction)

## How It Works
- User enters activity data
- Backend processes input and calculates carbon emissions
- Data is stored in MySQL
- Results are displayed on the dashboard with suggestions
